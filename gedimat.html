GEDIMA'GINATION

gedima'gination

La société NEGOMAT vient tout juste d'entrer dans le réseau d'adhérents de GEDIMAT.
Pour fêter cet événement, NEGOMAT a décidé d'organiser un jeu-concours nommé GEDIMA'GINATION.

Objectif et Problématique

L'objectif initial était d'avoir un site internet où les clients pourraient poster leur création.
Les votes se passeraient ensuite en magasin via une tablette.

Ainsi pour résumer :
- Un votant peut voter pour ses réalisations préferées
- Un participant peut participer au jeu-concours
- Un participant peut s'inscrire, se connecter, se déconnecter
- Un gestionnaire peut generer et poster les votes réaliser sur la tablette
- Un utilisateur peut voir les votes sans avoir de compte
- Un participant peut voir son classement en allant dans sont profil

Présentation

Pour mener à bien ce projet nous avons procédés par sprint avec different niveau de priorité.

- Documentation -

Inscription

Afin de pouvoir participer au jeu-concours, le participant doit ce crée un compte via ce formulaire.

Connexion

Apres l'inscription, le participant peux se connecter à son compte afin de pouvoir participer via un formulaire.

Le gestionnaire peut s'identifier afin de publier les résulats du concours.

Gestionnaire

Lorsque les votes sont terminé, le gestionnaire peux génerer et publier les resulats.

Les résultats

Lorsque le jeu-concours est terminé, les visiteurs peuvent voir les trois meilleurs réalisations.

Profil

Le participant peut aller voir son classement dans l'onglet "profil".

WEB

Une fois l'application démarée, le gestionnaire peu exporter la base, pour que celle-ci soit accessible a l'application web.

En appuyant sur "participer" un formulaire d'inscription s'ouvre.

Inscription votant

Pour pouvoir participer au vote, le votant doit s'inscrire via ce formulaire.

Un pop-up s'affiche si le votant ne coche pas la case consentement.

Vote

Apres s'être connecté, le votant peut voté pour ses trois réalisations préferés.

Un pop-up s'affiche si le votant vote pour la même réalisation plusieurs fois.
